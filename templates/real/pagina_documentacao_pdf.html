Valida Fácil
Sistema de Auditoria Semântica e LGPD
 Versão 1.0
O Valida Fácil utiliza um motor de análise híbrida para proteger dados sensíveis em sistemas de ouvidoria. A solução integra Processamento de Linguagem Natural (NLP) básico com regras heurísticas para impedir a exposição de dados pessoais.

1. Metodologia de Escaneamento

I. Motor Semântico (texto.py)
Identificação de intenções através da conjugação de verbos (solicitação vs. dúvida) e cruzamento com dicionário de termos sensíveis.
II. Validação Algorítmica
Implementação do Módulo 11 para filtragem de CPFs e CNPJs, eliminando falsos positivos antes do bloqueio.
III. Escala de Risco
Cálculo dinâmico de pontuação (Score 0.0 a 1.0) baseado na densidade de termos sensíveis e verbos de ação.

2. Matriz de Criticidade

Nível Intervalo Ação do Sistema
Inofensivo 0.0 - 0.2 Permite gravação direta no SQLite.
Moderado 0.2 - 0.6 Gera alerta visual (Amarelo) e log de auditoria.
Grave / Crítico 0.6 - 1.0 Bloqueio Preventivo e descarte de dados.

3. Arquitetura de Implementação

# Estrutura Principal de Auditoria
- Middleware: Filtro de requisições Flask (Blueprints)
- Motor: Singleton pattern para carga única de dicionários
- Persistência: Banco de dados SQLite com hash SHA-256
