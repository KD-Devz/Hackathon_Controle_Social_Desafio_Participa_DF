Documentação Técnica de Auditoria v1.0

1. Visão Geral

O sistema foi desenvolvido para atuar na interseção entre a Lei de Acesso à Informação (LAI) e a Lei Geral de Proteção de Dados (LGPD), automatizando a triagem de dados sensíveis em solicitações e-SIC.

2. Pipeline de Processamento

ETAPA 1 Sanitização de caracteres especiais.
ETAPA 2 Aplicação de expressões regulares para padrões PII.
ETAPA 3 Validação matemática de identificadores oficiais.
ETAPA 4 Reconhecimento de Entidades Nomeadas (NER) via NLP.

3. Guia de Deploy Rápido

pip install -r requirements.txt
flask db upgrade
python app.py
